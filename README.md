ssr.sh
------
Auto Install ShadowsocksR Server for CentOS/Debian/Ubuntu

Easy way:
```Shell
  $ wget -qO- http://tools.tisrop.com/ssr.sh | bash
```

Or by this way:
```Shell
  $ wget --no-check-certificate http://tools.tisrop.com/ssr.sh && chmod +x ssr.sh
  $ ./ssr.sh 2>&1 | tee ssr.log
```
