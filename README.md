* [ssr.sh](#ssrsh) - auto install ShadowsocksR


ssr.sh
------
Auto Install ShadowsocksR Server for CentOS/Debian/Ubuntu

Easy way:
```Shell
  $ bash <(curl http://tools.tisrop.com/ssr.sh)
```

Or by this way:
```Shell
  $ bash <(wget -O- http://tools.tisrop.com/ssr.sh)
```
